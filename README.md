# amazon-order-analysis.
This project focuses on analyzing Amazon sales data using SQL to identify key factors affecting order cancellations, product returns, and revenue performance. The goal of the analysis is to uncover patterns in customer behavior, product categories, payment methods, and geographic locations that contribute to business losses.

The dataset includes information about orders, customers, products, and payments. By performing SQL queries and data analysis, insights were generated such as identifying cities with the highest cancellation and return rates, analyzing product categories with frequent returns, and understanding customer purchasing behavior.

The project also evaluates which payment methods generate the highest revenue and highlights repeat customers who may be valuable for loyalty programs. Based on these insights, several data-driven recommendations were proposed to help businesses reduce return rates, improve customer satisfaction, and optimize revenue strategies.

Overall, this project demonstrates how SQL can be used to transform raw sales data into meaningful business insights that support better decision-making.
